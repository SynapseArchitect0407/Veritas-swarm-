Veritas Swarm is a multi AI agent AI system focused on distilling complex and unstructured data into accurate, aligned information thrpugh consensus mechanisms. Instead of relying on hallucinated single-model outputs, it emphasizes thinking accurately and validates truth by reducing hallucinations.Ultimately it moves AI close to being a trustworthy decision-making partner in critical fields like research, healthcare and global policy.
##Repository Structure
***core/**.Mathematical foundations(Tensors)
***agents/**.Individual researcher logic
***main.py**:The central orchestrator.
